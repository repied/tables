# Tables

A simple HTML+JS+CSS PWA app to plan dives using official MN90 tables.

## Install

## Contribute

* Open this folder in VS Code or in a Github Codespace.
* After the devcontainer is ready you can start a local development server with the installed `ms-vscode.live-server` extension, or with the included `npm run start`.  The application will be available at a local URL, typically `http://127.0.0.1:5500`.
* Playwright test can be run with `npm test`.
